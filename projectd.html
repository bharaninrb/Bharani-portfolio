Math Scribble Project

Web Design

About:

Math Scribble is an innovative web application designed to recognize and calculate handwritten mathematical expressions. Created using Figma and XD, this project employs advanced technologies such as TensorFlow, PyTorch, OpenCV, and SymPy for deep learning and symbolic mathematics. The fonts are sourced from Google Fonts, while images are obtained from Adobe Stock and Freepik. The primary goal is to enhance the accessibility and efficiency of mathematical expression processing, particularly benefiting students, educators, and visually impaired individuals.

Approach:

The approach involved developing deep learning models for object detection and character recognition, creating a responsive web interface using React or Vue.js, and integrating real-time calculation features. The design focused on accessibility and user experience, particularly for visually impaired users.

Challenges:
  • Training the app to understand complex handwritten equations took a lot of work, but it ensures your scribbles appear clean and clear on the screen.
  • Keeping up with your writing speed meant making the app super fast. Special tricks helped achieve this, ensuring a smooth UI experience.
  • The team didn't forget about users with visual impairments. Innovative design solutions made the UI usable for everyone.
  • Powerful tech powers Math Scribble, but the UI is still super friendly. They found the perfect balance between advanced features and a user-friendly interface.
  • Combining all this tech into a seamless app was a challenge. Constant testing and tweaks were crucial to create a smooth and enjoyable UI/UX.
Outcomes:

Math Scribble instantly transforms your messy math into clean digital calculations. No more waiting, it works in real-time! This user-friendly app even helps those with visual impairments tackle problems with ease. Math Scribble proves tech can be powerful and friendly, making math less frustrating. It's a leap forward for learning, opening doors to a more engaging future.
